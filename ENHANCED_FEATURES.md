# EdgeTunnel 增强版 - 时间限制与用户管理功能

## 项目概述

基于原版 EdgeTunnel（https://github.com/cmliu/edgetunnel）开发的增强版本，增加了时间限制和用户管理功能，特别适用于教育场景下的资源分享管理。

## 核心功能

### 1. 时间限制机制
- **真正的过期验证**：不仅显示到期时间，还实际阻止过期用户访问服务
- **动态订阅过期**：订阅内容中的过期时间根据用户实际到期时间动态生成
- **实时验证**：每次订阅请求都会验证用户有效性

### 2. 用户管理功能
- **创建用户**：通过管理API创建新用户并设置使用期限
- **用户续期**：支持为现有用户延长使用时间
- **用户删除**：移除不再需要的用户
- **用户列表**：查看所有注册用户及其状态
- **单个用户查询**：获取特定用户详细信息

### 3. 管理面板集成
- 与原有管理面板无缝集成
- RESTful API 设计，易于前端调用
- 完整的错误处理和状态反馈

## API 接口

### 用户管理接口

#### 创建用户
- **方法**: `POST`
- **路径**: `/admin/users/create`
- **请求体**:
```json
{
  "userId": "user_id", // 可选，默认使用系统生成的UUID
  "days": 30           // 使用天数，必填
}
```
- **响应**:
```json
{
  "success": true,
  "message": "用户创建成功",
  "user": {
    "id": "user_id",
    "created": 1234567890123,
    "expiration": 1234567890123,
    "days": 30
  }
}
```

#### 更新用户（续期）
- **方法**: `POST`
- **路径**: `/admin/users/update`
- **请求体**:
```json
{
  "userId": "user_id",
  "days": 60    // 新的使用天数
}
```

#### 删除用户
- **方法**: `POST`
- **路径**: `/admin/users/delete`
- **请求体**:
```json
{
  "userId": "user_id"
}
```

#### 获取用户列表
- **方法**: `POST`
- **路径**: `/admin/users/list`
- **响应**:
```json
{
  "success": true,
  "users": [
    {
      "id": "user_id",
      "created": 1234567890123,
      "expiration": 1234567890123,
      "days": 30,
      "expirationDate": "2023-12-31T23:59:59.000Z"
    }
  ]
}
```

#### 获取单个用户信息
- **方法**: `GET`
- **路径**: `/admin/users/{userId}`

## 部署要求

### 环境变量配置
- `USERS_KV` - KV命名空间绑定，用于存储用户数据

### 部署方式
- 支持通过ZIP文件上传部署到Cloudflare Pages
- 所有功能均可通过浏览器UI管理

## 安全特性

1. **严格验证**：每次订阅请求都进行用户有效性验证
2. **即时失效**：用户过期后立即无法访问服务
3. **独立存储**：用户数据与配置数据分离存储
4. **访问控制**：管理接口受原有权验证保护

## 使用场景

特别适用于：
- 教育机构内部资源共享
- 小团队协作工具访问控制
- 临时访问权限管理
- 非商业用途的资源分发

## 技术特点

- **零客户端修改**：无需修改客户端应用
- **向下兼容**：保留所有原有功能
- **轻量级实现**：最小化代码改动
- **高性能**：利用Cloudflare KV的低延迟特性

## 注意事项

- 需要配置相应的KV命名空间
- 建议定期备份用户数据
- 合理设置默认使用期限，避免过度分配
